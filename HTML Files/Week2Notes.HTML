Benefits to using semantic code:

- Makes it easier to Organize code

- Improves accesibility

Commonly used tags:

- The heading tag is commonly used to specify heading s in text. Referenced using the folowing format : H(number). Heading tags range in size from 1 to 6. Block level text element

- The paragraph tag normally follows the heading tag and us defined using the element P. Paragraph elements can appear one after another. Block level text element

The strong tag is used to bold text. There are two elements for bolding text: strong and b.

+ The strong element is semantically used to give strong importance to text and is the most popular.

+ The b element is semantically means to offset text, provide emphasis inline.

+ (ex. Strong text="caution". B text=I want you to"read" this.)

- The italicize tag, em, is use to place emphasis on text. There are two different elements that italicize text: em and i.

+ EM is used semantically to place a stressed emphasis on text.

+ i is used semantically to convey text in an alternative voice or tone, almost as if it were placed in quotation marks.

+ (ex. EM text=I"love" Chicago. I text=The name Shay means a gift.)
